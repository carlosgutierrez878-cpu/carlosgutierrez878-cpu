# DataMaster BI: Pipeline de Procesamiento y Visualización

Este proyecto es una herramienta robusta de **Ingeniería de Datos** diseñada para automatizar el flujo ETL (Extracción, Transformación y Carga) desde archivos planos hacia bases de datos relacionales, proporcionando una capa de visualización interactiva para la toma de decisiones.

## 🧠 Arquitectura del Sistema
El software opera bajo una arquitectura de tres capas:
1. **Capa de Ingesta:** Manejo dinámico de buffers para archivos CSV, Excel y JSON utilizando `Pandas`.
2. **Capa de Procesamiento:** Limpieza de datos (data cleaning), manejo de valores nulos y normalización de tipos de datos.
3. **Capa de Persistencia:** Mapeo de objetos a bases de datos (ORM) mediante `SQLAlchemy` para garantizar la integridad referencial en SQL.



## 🛠️ Características Técnicas
* **Motor de Gráficos:** Implementación de `Plotly Express` para renderizado de gráficos vectoriales interactivos.
* **Integración SQL:** Conexión flexible a motores de base de datos (SQLite/PostgreSQL) para persistencia de largo plazo.
* **Interfaz de Usuario:** Desarrollado con `Streamlit` para ofrecer una experiencia Web SPA (Single Page Application).
